# How To Learn (Лендинг)

Учебный сайт-лендинг от Яндекс.

## Технологии

- Семантическая структура html.
- Структура файлов, наименование классов в стиле BEM.
- Используется Flex.
- Несколько слоев фона.
- Используются CSS анимации.
